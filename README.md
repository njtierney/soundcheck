# soundcheck

<!-- badges: start -->

<!-- badges: end -->

The goal of soundcheck is to provide an interface into simple data
quality checking

## Installation

You can install the development version of soundcheck from
[GitHub](https://github.com/) with:

``` r
# install.packages("pak")
pak::pak("njtierney/soundcheck")
```

## Example

``` r
library(soundcheck)
run_app()
```
